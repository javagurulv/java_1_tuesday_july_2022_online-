WARNING: Не пишите тело метода в той же строке, что и его сигнатура.
Даже если тело метода занимает всего одну стоку.
Первая строка всегда предназначена для сигнатуры метода.
Она важна тому, кто будет использовать (вызывать) ваш метод,
так как из сигнатуры метода видно как называется метод,
какие параметры нужно передать в метод, что этот метод возвращает
в качестве результата. Тело метода в коде должно располагаться
на отдельной строке внутри фигурных скобок. Для тех, кому
важна сигнатура метода будут смотреть на первую строку, тем
кому интересна реализация метода будут смотреть на тело метода
во второй строке.

public boolean isEven(int number) {return (number % 2 == 0) ? true : false;}

Лучше:

public boolean isEven(int number) {
    return (number % 2 == 0) ? true : false;
}
